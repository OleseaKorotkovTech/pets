1. Скачайте OpenServer последней версии (6.4.6).

2. Включите OpenServer. В модулях включите MySql-8.4 и PHP-8.1.

3. Скачайте проект с Github.
   Если у вас есть Git, используйте
   git clone https://github.com/OleseaKorotkovTech/BreedsCatalog.git

    Если нет, скачайте архив и распакуйте его.

4. Переместите папку с проектом в папку OSPanel/home. Он должен появиться у вас в списке проектов.

5. Введите в браузерную строку phpmyadmin/. Убедитесь, что phpmyadmin включён в настройках OpenServer.

6. В phmyadmin выберите сервер MySql-8.4 и создайте базу данных с названием pets_db.

7. Создайте нового пользователя и выдайте ему все привилегии на созданную базу данных.

8. Внутри проекта найдите файл sql/pets_db.sql. Импортируйте его в только что созданную базу данных.

9. Внутри проекта откройте файл config/database.php и укажите там логин и пароль только что созданного пользователя.

10. В строке браузера введите pets/.
